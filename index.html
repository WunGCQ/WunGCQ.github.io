  • HTML5

    我相信HTML5所具有的种种特性将带领我们进入一个更加舒适、完美的视觉时代，多种新标签加强了语义化，canvas与webgl的使用让web更加丰富。

  • CSS3

    CSS3使内容排布更加合理、样式更加多变，让网页的交互充满了趣味！虽然各个浏览器对它的支持仍待加强，但是它绝对是至美页面不可或缺的一部分。

  • Javascript

    一个我无时无刻不在打交道的语言，拥有海量的框架、插件，独特的语法特性。从JQuery这种轻量的框架到YUI这种重型框架，Js始终是开发者的好伙伴

  • Qt

    C++的高度封装，完美的跨平台和运行效率，以及优秀的IDE。极品飞车就是Qt写的呢，So does PhotoShop

  • Java

    无需多言，这个星球上OOP最有统治力的语言，同样的跨平台，又是当下火热的Android应用开发语言。JVM也在不断优化，前途依旧光明。

私家珍藏，尚未上线，敬请期待~

在我的心中，优秀的编码者一定与音乐家有着某种深层的联系。
有本书叫《黑客与画家》，我倒认为，更像是音乐家。
用符号去书写思维,遵循着规则却又创造着规则，用外人看似枯燥与繁琐的符号去创造美。
在历史的潮流中，虽然有很多作品已经流逝，却总有一部分能穿越时光，让人们铭记。
音乐是一种态度，好的音乐总能折射出创作者的灵感、情感。
就像Scorpions这支乐队，40多年的高龄，但是他们的音乐总能打动我
是摇滚，是金属，但不止于感情的宣泄，而更多倾诉
总能带给我商业化的音乐所不能有的感动,是可以一个人静静欣赏的作品
好听，却不满足于好听。回想起来某一句歌词，总会脑海中浮现出旋律
让我相信，The best is yet to come！
